# Solution 2 — Product-formula error

If \([A,B]=0\), the exponential identity gives

\[
e^{-i(A+B)t/r}=e^{-iAt/r}e^{-iBt/r},
\]

so repeating the exact one-step equality \(r\) times gives exact total evolution.

For noncommuting groups, the asymptotic log-log slopes should approach approximately \(-1\) for first order and \(-2\) for second order when the selected range is within the asymptotic regime. Short-time, coarse-step, and floating-point regimes can distort fitted slopes.

At equal \(r\), second order uses more exponentials. A fair comparison chooses the least-cost parameter set for each method that satisfies the same target error.

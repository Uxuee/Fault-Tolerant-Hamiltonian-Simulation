# Exercise 2 — Product-formula error

1. Show that first-order Trotterization is exact when the grouped Hamiltonians commute.
2. Run the convergence example for \(t=0.25,1,4\).
3. Fit a straight line to \(\log(\text{error})\) versus \(\log(r)\) using the largest four values of \(r\).
4. Repeat for field strengths \(h=0.1,1,5\).
5. For each run, report both error and the naive number of arbitrary rotations.
6. Explain why comparing formulas at equal \(r\) is not the same as comparing them at equal error.
